# City of Coral Gables — Landing Page

Intended URL path: /for/city-of-coral-gables

A single-page, static, prepared read for the Innovation and Technology Department at the City of Coral Gables. The page mirrors the municipal site's civic register: Lora serifs for headings, Open Sans for body, forest-green masthead with the city wordmark, whitespace-led layout, photography-free composition. Content carries the fit-analysis pitch body verbatim, cites the Interpretable Context Methodology and Ethics Engine with live anchor links, and anchors the CTA on Matt Creamer's Calendly.

Files:
- `index.html` — the page
- `styles.css` — civic institutional CSS tokens from brand-pack.json
- `logo.png` — City of Coral Gables wordmark (from brand pack)

No JavaScript. Mobile responsive to 375px. External fonts loaded via Google Fonts (Lora + Open Sans) to match live municipal site typography. Prepared-by Eduba attribution appears only in the footer.
